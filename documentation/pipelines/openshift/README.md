# Deploy in OpenShift (PROD) with OpenShift Pipelines

Right Lyrics can be deployed in OpenShift using:

* OpenShift Pipelines
* Karpenter Tasks
* Kustomize Manifests

### Prerequisites

* OpenShift 4.2 or higher
* OpenShift Pipelines
* Tekton CLI (tkn)

### Usage

Run the [install.sh](install.sh) script.

```bash
sh install.sh
```
