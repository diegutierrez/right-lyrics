# Lyrics Page

## Local

```bash
export REACT_APP_LYRICS_SERVICE_URL=http://right.lyrics/api/lyrics/
export REACT_APP_SONGS_SERVICE_URL=http://right.lyrics/api/songs/
export REACT_APP_ALBUMS_SERVICE_URL=http://right.lyrics/api/albums/
export REACT_APP_KEYCLOAK_URL=http://right.lyrics/keycloak/auth/
export REACT_APP_KEYCLOAK_REALM=right-lyrics
export REACT_APP_KEYCLOAK_CLIENT_ID=lyrics-page
export REACT_APP_KEYCLOAK_ENABLED=true

npm start
```
